## Introduction

Remote Desktop Connection Manager (RDCMan) is a lightweight utility for organizing and launching multiple Remote Desktop (RDP) sessions from one place. Instead of juggling dozens of saved connections or opening separate windows for every server, you can keep your entire environment in a single, structured workspace. Connections are grouped into folders, so you can mirror how your infrastructure is arranged (by client, project, region, or role) and expand or collapse sections as needed.

RDCMan stores settings per group and per server, making it easy to apply common options—such as display size, gateway usage, redirection, or performance tweaks—while still allowing exceptions for specific machines. It supports saved credentials (with appropriate security precautions), quick reconnect, and easy switching between sessions, which is especially useful for administrators, support engineers, and anyone who routinely maintains Windows hosts.

The interface focuses on speed: double-click to connect, keep a grid of active sessions visible, and open properties to edit details without hunting through Windows dialogs. You can import existing RDP files, duplicate entries, and maintain separate connection sets for different tasks.
